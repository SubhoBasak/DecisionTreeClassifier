install libraries:

conda -

```bash
conda install numpy
conda install pandas
conda install matplotlib
conda install sklearn
conda install graphviz
```

```bash
pip install numpy
pip install pandas
pip install matplotlib
pip install sklearn
pip install graphviz
```

pip3 -

```bash
pip3 install numpy
pip3 install pandas

pip3 install sklearn

```
